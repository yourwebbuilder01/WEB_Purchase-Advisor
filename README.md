A smart web application that helps you make informed purchasing decisions by analyzing your financial situation and the value of potential purchases.

## Features

- **Smart Financial Analysis**: Calculates a purchase score based on multiple factors
- **Beautiful UI**: Soft, modern design with smooth animations
- **Usage-Based Value Assessment**: Considers how often you'll use the item
- **Financial Impact Analysis**: Shows effects on savings and emergency fund
- **Local Storage**: Saves your purchase history in the browser
- **Mobile Responsive**: Works perfectly on all devices

## How to Use

1. Enter the item you want to buy
2. Provide financial details (cost, savings, income, monthly spending)
3. Select how often you'll use the item
4. Get an instant analysis with recommendations

## Calculation Factors

The smart score considers:
- **Safety Score** (30%): Impact on emergency fund
- **Value Score** (30%): Cost per use over time
- **Affordability Score** (20%): Purchase price relative to income
- **Spending Score** (20%): Current spending habits
